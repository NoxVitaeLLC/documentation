# DJ Policies

---

## Primary DJ Selection

- Primary DJs for events are elected by collaborators ahead of the event, coordinated by DJ Logisticians.
- Primary DJs are given 2 weeks to test their own setup, our CDN (if differing from VRCDN), and streaming relays.
- If a Primary DJ does not complete testing within 1 week of the event, they will be replaced by a Backup DJ.

---

## Set Punctuality

- DJs are expected to be in-instance 5 minutes before their set.
- If a Primary DJ has not started playing within 2 minutes of their set start time, the stream will automatically be moved to a Backup DJ.
- We recommend joining the event VC to confirm you are in the correct instance — during peak hours multiple instances are typically active.

---

## Backup DJ Protocol

Backup DJs will stand in for any DJ who is unable to perform. This includes the following scenarios:

- Failure to appear 5 minutes prior to slot time
- Lack of communication within the DJ channel
- Missing or not attending scheduled DJ preparation checkpoints (remote stream test and in-venue stream test)
- Club rescinding a DJ nomination
- DJ rescinding booking
- DJ slot cancellation for any reason
- Any event that renders the DJ unable to stream audio into the venue
- Any additional reason rendering a DJ unable to fulfill the current booking
